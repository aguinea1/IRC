# GUÍA PARA PROBAR CANALES MANUALMENTE

## 1. Abrir terminal y conectarse al servidor:
nc localhost 6667

## 2. Registrarse (cliente 1 - Alice):
PASS testpass
NICK alice
USER alice 0 * :Alice User

## 3. En otra terminal, conectarse como cliente 2 (Bob):
nc localhost 6667
PASS testpass
NICK bob
USER bob 0 * :Bob User

## 4. Alice crea y se une al canal:
JOIN #test

## 5. Bob se une al mismo canal:
JOIN #test

## 6. Alice envía mensaje al canal:
PRIVMSG #test :Hola a todos!

## 7. Bob responde:
PRIVMSG #test :Hola Alice!

## 8. Alice establece tópico:
TOPIC #test :Canal de prueba

## 9. Bob consulta tópico:
TOPIC #test

## 10. Consultar usuarios del canal:
NAMES #test

## 11. Listar todos los canales:
LIST

## 12. Ver modos del canal:
MODE #test

## 13. Alice sale del canal:
PART #test :Adiós!

## 14. Salir del servidor:
QUIT
