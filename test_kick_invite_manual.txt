# GUÍA MANUAL PARA PROBAR KICK E INVITE
# ======================================

## PREPARACIÓN:
1. Abrir 3 terminales
2. En cada terminal: cd /Users/joannavarrogomez/Desktop/42/IRC
3. En terminal 1: ./ircserv 6667 testpass
4. En terminal 2: nc localhost 6667
5. En terminal 3: nc localhost 6667

## TERMINAL 2 - ALICE (OPERADOR):
```
PASS testpass
NICK alice
USER alice 0 * :Alice User
JOIN #test
```

## TERMINAL 3 - BOB (USUARIO NORMAL):
```
PASS testpass
NICK bob
USER bob 0 * :Bob User
JOIN #test
```

## PRUEBA 1: VERIFICAR QUE ALICE ES OPERADOR
En terminal 2 (Alice):
```
NAMES #test
```
Deberías ver: @alice bob (Alice con @ indica operador)

## PRUEBA 2: ALICE EXPULSA A BOB
En terminal 2 (Alice):
```
KICK #test bob :Por ser molesto
```
Deberías ver en ambos terminales:
- :alice!alice@127.0.0.1 KICK #test bob :Por ser molesto

## PRUEBA 3: BOB YA NO ESTÁ EN EL CANAL
En terminal 2 (Alice):
```
NAMES #test
```
Deberías ver solo: @alice

## PRUEBA 4: ALICE INVITA A BOB
En terminal 2 (Alice):
```
INVITE bob #test
```
Deberías ver:
- :alice!alice@127.0.0.1 INVITE bob :#test (en terminal 3)
- :irc.local 341 alice bob #test (en terminal 2)

## PRUEBA 5: BOB SE UNE (INVITADO)
En terminal 3 (Bob):
```
JOIN #test
```

## PRUEBA 6: VERIFICAR QUE BOB ESTÁ DE VUELTA
En terminal 2 (Alice):
```
NAMES #test
```
Deberías ver: @alice bob

## PRUEBA 7: BOB INTENTA EXPULSAR A ALICE (ERROR)
En terminal 3 (Bob):
```
KICK #test alice :No puedo hacer esto
```
Deberías ver error:
- :irc.local 482 bob #test :You're not channel operator

## PRUEBA 8: ALICE INTENTA EXPULSARSE A SÍ MISMA (ERROR)
En terminal 2 (Alice):
```
KICK #test alice :Me expulso
```
Deberías ver error:
- :irc.local 482 alice #test :You cannot kick yourself

## PRUEBA 9: ALICE INTENTA EXPULSAR A USUARIO INEXISTENTE
En terminal 2 (Alice):
```
KICK #test charlie :Usuario que no existe
```
Deberías ver error:
- :irc.local 441 charlie #test :They aren't on that channel

## PRUEBA 10: ALICE INVITA A USUARIO INEXISTENTE
En terminal 2 (Alice):
```
INVITE charlie #test
```
Deberías ver error:
- :irc.local 401 charlie :No such nick

## PRUEBA 11: ALICE INVITA A BOB QUE YA ESTÁ EN EL CANAL
En terminal 2 (Alice):
```
INVITE bob #test
```
Deberías ver error:
- :irc.local 443 bob #test :is already on channel

## PRUEBA 12: USUARIO NO REGISTRADO INTENTA KICK
En terminal 3 (Bob):
```
QUIT
```
Luego en terminal 3:
```
nc localhost 6667
KICK #test alice :Sin registrarse
```
Deberías ver error:
- :irc.local 451 * :You have not registered

## PRUEBA 13: USUARIO NO EN CANAL INTENTA KICK
En terminal 3:
```
PASS testpass
NICK charlie
USER charlie 0 * :Charlie User
KICK #test alice :No estoy en el canal
```
Deberías ver error:
- :irc.local 442 charlie #test :You're not on that channel

## COMANDOS DE LIMPIEZA:
En terminal 2:
```
QUIT :Test completado
```
En terminal 3:
```
QUIT :Test completado
```

## RESULTADOS ESPERADOS:
✅ KICK funciona solo para operadores
✅ INVITE funciona solo para operadores  
✅ Errores apropiados para casos inválidos
✅ Broadcast correcto de mensajes
✅ Validaciones de permisos
✅ Manejo de usuarios inexistentes
✅ Prevención de auto-expulsión

## CÓDIGOS DE ERROR IRC IMPLEMENTADOS:
- 401: No such nick
- 403: No such channel  
- 441: They aren't on that channel
- 442: You're not on that channel
- 443: is already on channel
- 451: You have not registered
- 461: Not enough parameters
- 482: You're not channel operator
- 341: RPL_INVITING (invitación exitosa)
